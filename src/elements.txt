ELEMENT EMPTY
NAME Air
COLOR 0,0,0

ELEMENT SAND
NAME Sand
COLOR 194,178,128
BEHAVIOR IF (0,1) EMPTY THEN SWAP(0,1)
BEHAVIOR IF (0,1) WATER THEN SWAP(0,1)

ELEMENT WATER
NAME Water
COLOR 0,0,255
BEHAVIOR IF (0,1) EMPTY THEN SWAP(0,1)
BEHAVIOR IF (-1,1) EMPTY THEN SWAP(-1,1)
BEHAVIOR IF (1,1) EMPTY THEN SWAP(1,1)
